# Changelog - Visuome

All notable changes to the Visuome design intelligence platform are documented here.

## [0.5.1] - 2026-07-06
### Added
- Added `.gitignore` and `.npmignore` release configurations.
- Added GitHub Actions pipeline validation workflow `build.yml`.
- Added build verification script `scripts/validateBuild.mjs`.
- Added health evaluation script `scripts/healthCheck.mjs`.
- Added `ArchitectureValidator.js` verifying registries and pipeline stages sequence.
- Expanded diagnostics to trace visited/skipped DOM elements, style cache hits/misses, registry sizes, and slowest stages timings.

## [0.5.0] - 2026-07-06
### Added
- Created independent Motion & Interaction Intelligence subsystem (Volume 4).
- Added five browser-native motion discovery modules (Animations, Transitions, Scroll, Timelines, Interaction).
- Added metrics calculators and triggers detectors.
- Added safe isolated technology scanner identifying GSAP, AOS, ScrollReveal, and WAAPI.
- Added motion semantics and motion philosophy mapping layers.

## [0.4.0] - 2026-07-06
### Added
- Implemented Visual Language, Semantic Intelligence, and Design Philosophy stages (Volume 3).
- Added specialized characteristics and philosophy analyzers.
- Wired pipeline sequences and registered mappings inside final reports.

## [0.3.0] - 2026-07-06
### Added
- Implemented core foundation discovery stages (Volume 2).
- Added node registry, visual tree builder, style cataloger, and token extractors.

## [0.1.0] - 2026-07-06
### Added
- Initial project MVP architecture and design tokens (Volume 0 & 1).
